# 🚀 Smart Productivity Dashboard 🎯

A modern and responsive Productivity Dashboard built using HTML, CSS, and JavaScript to help users manage tasks and track daily habits efficiently.

---

## ✨ Features

### 📋 Task Management
- Add, edit, and delete tasks  
- Mark tasks as completed ✔  
- Priority levels (High / Medium / Low)  
- Due date tracking 📅  
- Overdue & today task highlighting  

---

### 🔍 Smart Filtering
- Filter tasks:
  - All  
  - Completed  
  - Pending  
- Filter by priority  
- Sorted by priority automatically  

---

### 📊 Analytics Dashboard
- Task status visualization (Completed vs Pending)  
- Priority distribution chart  
- Clean and minimal insights  
- Built using Chart.js  

---

### 🌙 Dark / Light Mode
- Toggle between themes  
- Smooth UI switching  
- Theme saved using localStorage  

---

### 🔁 Habit Tracker 🔥
- Add daily habits  
- Mark habits as completed ✔  
- Track streaks 🔥  
- Simple and effective UI  

---

### 📱 Responsive Design
- Works on:
  - Mobile 📱  
  - Tablet 💻  
  - Desktop 🖥  
- Sidebar with hamburger menu  

---

## 🛠 Tech Stack

- HTML5  
- CSS3 (Flexbox + Grid)  
- JavaScript (ES6 Modules)  
- Chart.js  
- LocalStorage  

---

## 🎨 UI Highlights

- Clean and modern dashboard design  
- Soft color palette for readability  
- Card-based layout  
- Dark mode optimized UI
